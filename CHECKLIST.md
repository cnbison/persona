# 开发检查清单（Persona方向）

> **最新更新**: 2026-02-04
> **说明**: 本清单聚焦“环境准备 + 质量验收 + 发布准备”，开发任务请以 `TASKS.md` 为准。

---

## 一、环境准备

- [ ] Python 3.10+ 已安装
- [ ] Node.js 18+ 已安装
- [ ] Rust/Tauri 环境已安装（桌面端）
- [ ] `.env` 已配置 OpenAI API Key
- [ ] 依赖安装完成（后端/前端）

---

## 二、证据库与可追溯性

- [ ] 文档解析成功率达标（≥98%）
- [ ] 观点-原文-上下文证据链完整
- [ ] 证据库可检索、可定位

---

## 三、说者Persona构建

- [ ] Persona画像可解释（默认六维度或自定义维度）
- [ ] 关键观点/事实证据匹配率100%
- [ ] Persona卡片可编辑、可追溯
- [ ] Persona配置可导出/导入（JSON）

---

## 四、受众Persona适配

- [ ] 受众Persona覆盖小学/初中/高中/大学/成人
- [ ] 多版本输出可比较（Diff）
- [ ] 概念/事实锁定生效
- [ ] 受众适配评估（理解题/可读性）可用

---

## 五、可解释性与诊断

- [ ] 三阶段输出（canonical/plan/final）
- [ ] 诊断提示可行动（术语过密/抽象过高/偏离证据）
- [ ] 风格热力图或可视化输出可用

---

## 六、发布准备

- [ ] 关键流程回归测试通过
- [ ] 成本与性能指标在可接受范围
- [ ] 合规与版权检查流程完成
- [ ] 打包/发布流程可执行
